PCA

神经网络本质的目的是为了检查和生成来源于客观世界的信息。客观世界的信息，从一张图片到一段文字，是一个极大的数量级。考虑一张 28×28 的纯黑白二值图片，那么整个数据的空间大小就是 228×28。如果我们能给这里面的每张图片都生成一个标签，那么我们显然可以准确无误地对所有这个尺寸的图片做判断。

但是，显然现实中我们不可能得到所有的图片数据，更不可能储存起来。我们考虑这个问题：在给定储存空间下，我们如何最好地利用储存空间，使得我们储存的数据更逼近原来的数据呢？

问题的形式化

假设我们有 𝑥⃗1,𝑥⃗2,…,𝑥⃗𝑁;𝑥⃗𝑖∈ℝ𝑀，或者说矩阵 X∈ℝ𝑁×𝑀 的数据。我们想要得到一个子空间，其基向量为：

𝑏 ⃗ 1 , 𝑏 ⃗ 2 , … , 𝑏 ⃗ 𝑚 ; 𝑏 ⃗ 𝑖 ⋅ 𝑏 ⃗ 𝑗 = 0 ( 𝑖 ≠ 𝑗 ) ; ‖ 𝑏 ⃗ 𝑖 ‖ 2 = 1

也可以写成正交矩阵 B∈ℝ𝑚×𝑀。

对应的，𝑧⃗1,𝑧⃗2,…,𝑧⃗𝑁∈ℝ𝑚 为原数据近似到这个子空间的低维坐标，同时：

𝜒 ⃗ 𝑖 = B 𝑇 𝑧 ⃗ 𝑖 ∈ ℝ 𝑀

也就是在原高维空间中近似 𝑥⃗𝑖 的重构向量。

其中 𝑚≪𝑀≪𝑁，我们只需要储存基矩阵 B 和低维坐标 𝑧⃗𝑖,(𝑖=1,2,…,𝑁)，从而大幅减少数据量。

如何衡量这个压缩的效果呢？或者说，怎么样才能让重构数据和原数据更逼近呢？

投影

我们先不在意基向量 𝑏⃗𝑖 的具体选择，而是考虑在确定的 B 下，𝜒⃗𝑖 和 𝑥⃗𝑖 的关系。也就是说，如何选取一组在 B 下的坐标，使得信息损失最少（两者的欧氏距离最小）？

形式化地，也就是求以下目标函数的最小值：

𝐿 = 1 𝑁 ∑ 𝑖 = 1 𝑁 ‖ 𝑥 ⃗ 𝑖 − 𝜒 ⃗ 𝑖 ‖ 2 , 𝜒 ⃗ 𝑖 = B 𝑇 𝑧 ⃗ 𝑖

此时的自变量是低维坐标 𝑧⃗𝑖。这里使用分母布局直接对 𝑧⃗𝑖 求导：

𝜕 𝐿 𝜕 𝑧 ⃗ 𝑖 = 𝜕 𝜒 ⃗ 𝑖 𝜕 𝑧 ⃗ 𝑖 𝜕 𝐿 𝜕 𝜒 ⃗ 𝑖 = 2 𝑁 B ( 𝑥 ⃗ 𝑖 − 𝜒 ⃗ 𝑖 ) = 0 ⇒ B 𝑥 ⃗ 𝑖 = B 𝜒 ⃗ 𝑖 = B B 𝑇 𝑧 ⃗ 𝑖

因为 B 的各行是标准正交基，故 BB𝑇=I𝑚，因此我们得到了：

𝑧 ⃗ 𝑖 = B 𝑥 ⃗ 𝑖

也就是说，对特定的基矩阵 B，使得损失最小的办法就是直接将原向量投影到 B 支撑的子空间。

角度一: 最大的多样性

假设 𝑀=2,𝑚=1，考虑如左图所示的数据集。显然如果我们选择 𝑏⃗1 为右图的蓝线方向来近似，能得到最好的效果。

为什么呢？我们既可以说这些点和这条线贴合得最近，也可以说所有的数据在这个子空间（也就是这条线）上投影后伸展得最开，也就是说，展现了最大的多样性（方差最大）。反之，如果选择和蓝线正交的方向，所有的点在投影后会紧密堆积在一起，失去原有的区分度。

为了方便计算，我们可以先假设数据已经过了中心化预处理，即均值为 0：

𝔼 = 1 𝑁 ∑ 𝑖 = 1 𝑁 𝑥 ⃗ 𝑖 = 0 ⃗

因此目前的方差等价于

𝕍 𝑁 = 1 𝑁 ∑ 𝑖 = 1 𝑁 𝑧 ⃗ 𝑖 𝑇 𝑧 ⃗ 𝑖 ∈ ℝ

.

因此

𝕍 𝑁 = 1 𝑁 ∑ 𝑖 = 1 𝑁 𝑧 ⃗ 𝑖 𝑇 𝑧 ⃗ 𝑖 = 1 𝑁 ∑ 𝑖 = 1 𝑁 tr ( 𝑧 ⃗ 𝑖 𝑧 ⃗ 𝑖 𝑇 ) = 1 𝑁 ∑ 𝑖 = 1 𝑁 tr ( B 𝑥 ⃗ 𝑖 𝑥 ⃗ 𝑖 𝑇 B 𝑇 )

因为求和和求迹都是线性算子，把求和号移进迹

= 1 𝑁 tr ( B ( ∑ 𝑖 = 1 𝑁 𝑥 ⃗ 𝑖 𝑥 ⃗ 𝑖 𝑇 ) B 𝑇 ) = 1 𝑁 tr ( B S B 𝑇 )

S是𝑥不同分量的协方差矩阵, 因此是对称的, 所以有

= 1 𝑁 tr ( B U Λ U 𝑇 B 𝑇 ) = 1 𝑁 tr ( P Λ P 𝑇 ) , P = BU

我们来检查新矩阵 P∈ℝ𝑚×𝑀 的性质。由于 B 满足正交约束 BB𝑇=I𝑚，且 U 是正交阵（UU𝑇=I𝑀），因此有：

P P 𝑇 = ( B U ) ( B U ) 𝑇 = B U U 𝑇 B 𝑇 = B I 𝑀 B 𝑇 = B B 𝑇 = I 𝑚

这说明新矩阵 P 的这 𝑚 个行向量之间依然保持着标准正交的关系。

现在，我们将目标函数的矩阵迹展开为对角阵的加权求和形式：

1 𝑁 tr ( P Λ P 𝑇 ) = 1 𝑁 ∑ 𝑗 = 1 𝑚 ∑ 𝑘 = 1 𝑀 𝑃 𝑗 , 𝑘 2 𝜆 𝑘 = ∑ 𝑘 = 1 𝑀 ( ∑ 𝑗 = 1 𝑚 𝑃 𝑗 , 𝑘 2 ) ( 𝜆 𝑘 𝑁 )

令 𝑤𝑘=∑𝑗=1𝑚𝑃𝑗,𝑘2 作为分配给第 𝑘 个特征值的“权重配额”。由于 P 的行向量是标准正交的，这些配额受到两个严格的边界限制：

  1. 全局总配额有限：∑𝑘=1𝑀𝑤𝑘=𝑚（即矩阵 P 所有元素的平方和等于其行数 𝑚）。
  2. 单项上限约束：由于 P 的列向量可以看作U的一个模长为1的向量在B子空间的投影坐标, 因此其平方和是模长, 投影模长不可能超过原始模长1，故对任意 𝑘 均有 0≤𝑤𝑘≤1。

此时，最大化总方差的问题演变成了一个经典的贪心策略问题：我们的手里一共有总和为 𝑚 的筹码，每个坑位最多只能填 1 个筹码，而每个坑位的奖励权重是降序排列的（𝜆1≥𝜆2≥…≥𝜆𝑀≥0, S是半正定的）。

为了让最终的加权和最大，最完美的策略显然是将所有的筹码配额优先塞满奖励最高的前 𝑚 个坑位，即令：

𝑤 1 = 1 , 𝑤 2 = 1 , … , 𝑤 𝑚 = 1 ; 𝑤 𝑚 + 1 = … = 𝑤 𝑀 = 0

所以

𝕍 𝑁 = ∑ 𝑘 = 1 𝑀 ( ∑ 𝑗 = 1 𝑚 𝑃 𝑗 , 𝑘 2 ) ( 𝜆 𝑘 𝑁 ) ≤ 1 𝑁 ( 𝜆 1 + … + 𝜆 𝑚 )

而如果我们取B的行向量是这些特征值对应的特征向量, 也就是

S 𝑏 ⃗ 𝑖 = 𝜆 𝑖 𝑏 ⃗ 𝑖 , 𝑖 = 1 , 2 , … , 𝑚

那么有

P = B U = ( 𝑏 ⃗ 1 𝑏 ⃗ 2 … 𝑏 ⃗ 𝑚 ) ( 𝑏 ⃗ 1 𝑏 ⃗ 2 … 𝑏 ⃗ 𝑖 … 𝑏 ⃗ 𝑀 ) = ( I 𝑚 0 )

这里𝑏⃗𝑖的序数被扩大了, 表示未被选中的特征向量, 仍然按照特征值的大小排序。 这个时候就是

𝑤 1 = 1 , 𝑤 2 = 1 , … , 𝑤 𝑚 = 1 ; 𝑤 𝑚 + 1 = … = 𝑤 𝑀 = 0

所以这个不等号可以被取到. 这样我们就证明了最大的值在我们选择B的前𝑚大的特征值的特征向量时取到.注意， 这个 并非唯一的解， 只要B的子空间是一样的就可以了。 同时，约定 X 的 SVD 为 X=VΣU𝑇（其中 U∈ℝ𝑀×𝑀 为右奇异向量矩阵），则：

S = X 𝑇 X = ( U Σ V 𝑇 ) ( V Σ U 𝑇 ) = U Σ 2 U 𝑇

因此为了求S的特征值, 我们只需要对X做SVD即可.

角度二: 最小的损失

推导出PCA不只有一种角度, 在所有的角度上, 推出PCA似乎都是一种必然. 我们考虑这样的事情, 不再考虑子空间舒展的多样性, 而是考虑最小化和原向量的误差, 也就是

𝐽 = ∑ 𝑖 = 1 𝑁 ‖ 𝑥 ⃗ 𝑖 − 𝜒 ⃗ 𝑖 ‖ 2

同样的, 为了方便计算，我们可以先假设数据已经过了中心化预处理，即均值为 0：

𝔼 = 1 𝑁 ∑ 𝑖 = 1 𝑁 𝑥 ⃗ 𝑖 = 0 ⃗

我们对𝑏⃗𝑖做施密特正交化的扩展， 得到

𝑏 ⃗ 1 , 𝑏 ⃗ 2 , … , 𝑏 ⃗ 𝑀 ; 𝑏 ⃗ 𝑖 ⋅ 𝑏 ⃗ 𝑗 = { 1 𝑖 = 𝑗 0 𝑖 ≠ 𝑗

因此我们有

𝜒 ⃗ 𝑖 = ∑ 𝑗 = 1 𝑚 𝑧 𝑖 𝑗 𝑏 ⃗ 𝑗

而

𝑥 ⃗ 𝑖 = ∑ 𝑗 = 1 𝑀 𝑧 𝑖 𝑗 𝑏 ⃗ 𝑗

这里就是𝑥⃗𝑖变化到了𝑏⃗𝑗这个完整空间的完整坐标。因为𝑧⃗𝑖=B𝑥⃗𝑖，𝜒⃗𝑖就是前𝑚项内容在高维空间的重构，我们对𝑧⃗𝑖长度扩充的话，就是𝑥⃗𝑖的完整坐标. 因此

𝐽 = ∑ 𝑖 = 1 𝑁 ‖ 𝑥 ⃗ 𝑖 − 𝜒 ⃗ 𝑖 ‖ 2 = ∑ 𝑖 = 1 𝑁 ( ∑ 𝑗 = 𝑚 + 1 𝑀 𝑧 𝑖 𝑗 𝑏 ⃗ 𝑗 ) 2

发现了吗？ 我们得到了和角度一形式上完全对偶的一个式子——只不过这里的B是B′，也就是 剩下的向量，这里的𝑧⃗𝑖是剩下的向量的项。相对的，求最大值变成了最小值。我们前面分析 了，值其实是对𝜆𝑖的选择，那么为了最小化，我们选择最小的𝜆𝑖. 那么这个时候，原始B的空间其实就是𝜆1,𝜆2,…,𝜆𝑚长成 的子空间。而这是我们可以更直观地发现，其实B不必然是𝜆𝑖对应的特征向量，我们只需要 找到这个最大逼近的子空间就可以了！不过显然做SVD找起来最简单方便的。

角度三： 信息熵逼近

从信息论的角度来看，高维数据中包含的信息量可以用信息熵（Information Entropy）来衡量。对于多元高斯分布的数据，其熵的大小直接正比于其协方差矩阵行列式的对数。

假设高维原始数据 𝑥⃗𝑖∈ℝ𝑀 服从均值为 0⃗，协方差矩阵为 S 的多元正态分布 𝒩︀(0⃗,S)。其连续信息熵定义为：

𝐻 ( X ) = − 𝔼 [ ln 𝑝 ( 𝑥 ⃗ ) ] = − 𝔼 [ ln ( 1 ( 2 𝜋 ) 𝑀 2 det ( 𝑆 ) 1 2 exp ( − 1 2 ( 𝑥 ⃗ − 𝜇 ⃗ ) 𝑇 S − 1 ( 𝑥 ⃗ − 𝜇 ⃗ ) ) ) ] = 1 2 ln ( ( 2 𝜋 ) 𝑀 det ( S ) ) + 1 2 𝔼 [ ( 𝑥 ⃗ − 𝜇 ⃗ ) 𝑇 S − 1 ( 𝑥 ⃗ − 𝜇 ⃗ ) ] = 1 2 ln ( ( 2 𝜋 ) 𝑀 det ( S ) ) + 1 2 𝔼 [ tr ( ( 𝑥 ⃗ − 𝜇 ⃗ ) 𝑇 S − 1 ( 𝑥 ⃗ − 𝜇 ⃗ ) ) ] = 1 2 ln ( ( 2 𝜋 ) 𝑀 det ( S ) ) + 1 2 𝔼 [ tr ( ( 𝑥 ⃗ − 𝜇 ⃗ ) ( 𝑥 ⃗ − 𝜇 ⃗ ) 𝑇 S − 1 ) ] = 1 2 ln ( ( 2 𝜋 ) 𝑀 det ( S ) ) + 1 2 𝔼 [ tr ( S S − 1 ) ] = 1 2 ln ( ( 2 𝜋 ) 𝑀 det ( S ) ) + 1 2 𝑀 = 𝑀 2 ( 1 + ln ( 2 𝜋 ) ) + 1 2 ln det ( S )

当我们将数据投影到由 B∈ℝ𝑚×𝑀 支撑的低维子空间，得到 𝑧⃗𝑖=B𝑥⃗𝑖 时，由前面的推导， 低维变量 𝑧⃗𝑖 的协方差矩阵变为 S𝑧=BSB𝑇。

此时，低维空间所保留的信息熵为：

𝐻 ( Z ) = 𝑚 2 ( 1 + ln ( 2 𝜋 ) ) + 1 2 ln det ( BS B 𝑇 )

为了让压缩后的低维特征最大程度地保留原始高维空间的信息（即信息流失最少），我们的目标是最大化低维空间的熵 𝐻(Z)。在基向量标准正交（BB𝑇=I𝑚）的约束下，这等价于求解：

max B det ( BS B 𝑇 ) subject to B B 𝑇 = I 𝑚

利用对称阵 S 的特征值分解 S=UΛU𝑇，并令 P=BU（满足 PP𝑇=I𝑚），目标矩阵可以写为：

BS B 𝑇 = P Λ P 𝑇

由 Cauchy-Binet 公式，对任意满足 PP𝑇=I𝑚 的矩阵 P∈ℝ𝑚×𝑀 与半正定对角阵 Λ，有：

det ( P Λ P 𝑇 ) = ∑ | 𝑆 | = 𝑚 ∏ 𝑘 ∈ 𝑆 𝜆 𝑘 ⋅ 𝑚 𝑆 2 ≤ ∏ 𝑖 = 1 𝑚 𝜆 𝑖

其中求和遍历所有大小为 𝑚 的下标子集 𝑆，𝑚𝑆 为 P 对应列构成的 𝑚×𝑚 子方阵的行列式，且由 Cauchy-Binet 公式知 ∑|𝑆|=𝑚𝑚𝑆2=det(PP𝑇)=1。因此上界等号当且仅当所有权重集中在 𝑆={1,…,𝑚}（对应最大 𝑚 个特征值）时取到。

当且仅当 P=(I𝑚0) 时，等号成立。此时 B 对应的行向量恰好是 S 前 𝑚 个最大特征值所对应的特征向量。

角度四：线性自编码器的等价性

回到文章开头的问题：神经网络能从数据中学到什么？我们来考察最简单的一种神经网络——线性自编码器。它由两个无激活函数的线性层组成：

encoder 𝑧 ⃗ 𝑖 = W 𝑒 𝑥 ⃗ 𝑖 , W 𝑒 ∈ ℝ 𝑚 × 𝑀 decoder 𝜒 ⃗ 𝑖 = W 𝑑 𝑧 ⃗ 𝑖 = W 𝑑 W 𝑒 𝑥 ⃗ 𝑖 , W 𝑑 ∈ ℝ 𝑀 × 𝑚

训练目标是最小化重构误差：

𝐿 ( W 𝑒 , W 𝑑 ) = 1 𝑁 ∑ 𝑖 = 1 𝑁 ‖ 𝑥 ⃗ 𝑖 − W 𝑑 W 𝑒 𝑥 ⃗ 𝑖 ‖ 2

与PCA的角度二对比，这个损失函数的形式完全相同，区别仅在于约束条件：PCA要求 BB𝑇=I𝑚，而线性自编码器对 W𝑒 和 W𝑑 没有任何正交约束。

这看起来是一个更宽松的问题——那么它的解还是PCA吗？

参数空间的等价重构

令 A=W𝑑W𝑒∈ℝ𝑀×𝑀，注意到：

rank ( A ) = rank ( W 𝑑 W 𝑒 ) ≤ min ( rank ( W 𝑑 ) , rank ( W 𝑒 ) ) ≤ 𝑚

反过来，任意一个秩不超过 𝑚 的矩阵 A∈ℝ𝑀×𝑀 都可以通过秩分解写成 A=W𝑑W𝑒 的形式。因此，对 (W𝑒,W𝑑) 的联合优化，与直接对所有秩不超过 𝑚 的矩阵 A 优化，是完全等价的：

min W 𝑒 , W 𝑑 𝐿 ( W 𝑒 , W 𝑑 ) = min A , rank ( A ) ≤ 𝑚 1 𝑁 ∑ 𝑖 = 1 𝑁 ‖ ( I − A ) 𝑥 ⃗ 𝑖 ‖ 2

右边的问题正是角度二中的最小重构误差问题，只不过去掉了正交性约束，允许 A 是任意秩为 𝑚 的矩阵（而不仅仅是正交投影）。

然而，从角度二的推导中我们已经知道：对任意固定子空间，使重构误差最小的映射恰好是正交投影。换句话说，即使允许 A 是任意秩-𝑚 矩阵，最优的 A∗ 也必然是某个 𝑚 维子空间上的正交投影算子：

A ∗ = U 𝑚 U 𝑚 𝑇

其中 U𝑚 的列为 S 前 𝑚 个最大特征值对应的特征向量。这与角度二的结论完全吻合——PCA 子空间是全局最优解。

一组具体的最优参数为 W𝑑=U𝑚,W𝑒=U𝑚𝑇，但最优解并不唯一：对任意 𝑚×𝑚 正交矩阵 R，令

W 𝑑 = U 𝑚 R , W 𝑒 = R 𝑇 U 𝑚 𝑇

同样满足 W𝑑W𝑒=U𝑚U𝑚𝑇，因此也是全局最优解。这说明线性自编码器的权重矩阵本身存在旋转自由度：网络学到的编码和解码方向在子空间内可以任意旋转，但它们共同张成的子空间始终是唯一确定的 PCA 主成分子空间。

一个没有任何正交约束、通过梯度下降训练的线性神经网络，在全局最优时自然地发现了数据中最重要的线性结构。一方面， 他说明了神经网络真正的能够总结规律， 另一方面， 也说明 了PCA就是提炼规律的最佳结果。

总结

我们从四个完全不同的角度出发，推导出了同一个结论：

角度 优化目标 最优解
最大多样性 最大化投影方差 tr(BSB𝑇) 前 𝑚 大特征向量
最小损失 最小化重构误差 ∑‖𝑥⃗𝑖−𝜒⃗𝑖‖2 前 𝑚 大特征向量
